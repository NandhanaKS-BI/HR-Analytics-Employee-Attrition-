# 📊 HR Analytics – Employee Attrition Dashboard (Power BI)

## 📌 Project Overview
This project analyzes employee attrition data to identify key factors influencing employee turnover. 
The dashboard provides insights into workforce trends, attrition patterns, and HR metrics using interactive visualizations built in Power BI.

---

## 🎯 Objective
The objective of this project is to:
- Analyze employee attrition trends
- Identify key factors contributing to employee turnover
- Evaluate department-wise and role-wise attrition
- Support HR decision-making using data insights

---

## 🛠 Tools & Technologies Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Cleaning & Transformation (Power Query)
- Excel (Data Source)

---

## 📊 Key Metrics (KPIs)
- Total Employees
- Total Attrition Count
- Attrition Rate (%)
- Active Employees
- Average Age
- Average Salary
- Average Years at Company

---

## 📈 Dashboard Analysis Includes
- Attrition by Department
- Attrition by Job Role
- Attrition by Gender
- Attrition by Age Group
- Attrition by Education Field
- Salary vs Attrition Analysis
- Years at Company vs Attrition

---

## 🔎 DAX Calculations Used
- Attrition Rate Calculation
- Active Employees Count
- Average Salary
- Employee Count by Category
- Custom calculated measures for HR KPIs

---

## 📊 Dashboard Features
- Interactive slicers (Department, Gender, Education, etc.)
- KPI summary cards
- Dynamic filtering
- Drill-down analysis
- Clean and professional layout
